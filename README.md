# 1. Cài đặt và chạy source code
## Node version
Install NVM and using Node.js 12.22.12
```bash
nvm use default v12.22.12
```

## Install library
```bash
yarn install
```
## Start

### Start APP

```bash
yarn start
```
### Start on andoird device / ios device
```bash
yarn android
```
or
```bash
yarn ios
```
